phyto-water-level-change-review

This markdown:

  1. reads in the extracted data from the systematic literature review.
  2. applies 1_qaqc_data_sysreview.R to exclude any sites that were erroneously extracted, to de-duplicate any sites/data that were included in this review in multiple studies, and to fix assorted typos and mistakes in the raw extracted data.
  3. applies 2_extract_drivers.R to create new columns for phytoplankton and cyanobacterial drivers, responses, and measured variables.
  4. applies 3_proportion_WL.R to calculate proportional change in water level for the logistic regression.
  5. creates all of the manuscript figures.
  6. creates all of the supplemental information (SI) figures.

extract_cellwise_posthoc <- function(tab, model_name, p.adjust.method = "bonferroni") {

  # run global chi-square with descriptors
  chi_res <- tab %>% 
    chisq_test() %>%
    chisq_descriptives() %>%
    #std_residuals() %>%        # standardized residuals
    dplyr::select(water_direction_final, response_final, observed, expected, std.resid)

  # compute two‑sided p value from standardized residuals
  chi_res <- chi_res %>%
    mutate(p_raw = 2 * (1 - pnorm(abs(std.resid))))

  # multiple test correction
  chi_res <- chi_res %>%
    mutate(p_adj = p.adjust(p_raw, method = "bonferroni"),
           model = model_name)

  return(chi_res)
}
